## Sobre o Bootcamp

<br>

<p>O Bootcamp Fullstack Santander é uma iniciativa para dessiminar conhecimento através da distribuição de bolsas de estudos a milhares de estudantes e profissionais da área de tecnologia, área esta que está defasada de bons profissionais.</p>

<p>Neste Bootcamp foram abordadas as seguintes tecnologias e conceitos:</a>

* Programação e Pensamento Computacional;
* Introdução ao Git e ao GitHub;
* HTML5;
* CSS3;
* Javascript;
* Typescript;
* Angular8;
* Java;
* Programação Orientada a Objetos;
* Spring Framework com Spring Boot;
